interval quality

As you may have already noticed, the interval number doesn't give us all the information about an interval.

For example:

  • A-B is a second. B-C is also a second, but B-C is a semitone, whereas A-B is a tone. The interval A-B is therefore bigger than B-C.
  • The note C♯ is higher than the note C, so the interval A-C♯ is bigger than A-C, but both are thirds. (Remember that to find the interval number we only looked at the number of stave positions.)
  • C-E and D-F are both thirds, but C-E is made up of two tones, whereas D-F is made up of a tone and a semitone.

In addition to the interval number, to fully describe an interval, we will need something else called interval quality.

To describe the interval quality, we use the words major, minor, perfect, augmented and diminished. Once we have found the number and quality of an interval, we have identified the interval.

We will see that some intervals (namely 2nds, 3rds, 6ths and 7ths) can be major or minor. The major interval will be bigger than the minor interval by one semitone. If we made the major interval even bigger by one semitone, it would be augmented. If we made the minor interval even smaller by one semitone, it would be diminished.

Other intervals (unisons, 4ths, 5ths and octaves) can be perfect (these can't be major or minor). If we made a perfect interval one semitone bigger, it would be augmented. If we made a perfect interval one semitone smaller, it would be diminished.

We can now draw the following chart to summarise the meaning of these words.

2nds, 3rds, 6ths, 7ths
diminished minor major augmented
unisons, 4ths, 5ths, 8ves
diminished perfect augmented
← smaller ------ bigger →

Look at the examples below and check that they all follow these rules. We haven't yet learned how to identify intervals, but, you can check that the examples below follow the rules stated above.

C-E is a major 3rd C-E♭ is a semitone smaller, so it is a minor 3rd C♯ - E is also a semitone smaller than the major 3rd, so it is also a minor 3rd
F - G is a major 2nd F♯ - G is therefore a
minor 2nd		 F -G♯ is a semitone bigger than the major 2nd, so it is an augmented 2nd
C - G is a perfect 5th C ♯ - G is therefore a
diminished 5th		 C -G♯ is a semitone bigger than the perfect 5th, so it is an augmented 5th

If this is new to you and seems very confusing, don't worry! You will soon become familiar with these terms and with practice, you will soon be able to identify any interval.
