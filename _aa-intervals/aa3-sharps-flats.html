what if there are sharps and flats?

what we've learnt so far

In the last section we learnt to identify all the intervals between notes in the C major scale. In the section before that, we learnt about the words major, minor, perfect, augmented and diminished. Here is the chart again showing what they mean.

2nds, 3rds, 6ths, 7ths
diminished minor major augmented
unisons, 4ths, 5ths, 8ves
diminished perfect augmented
← smaller ------ bigger →

Now we can put together everything we've learnt and this will allow us to identify any interval.

If we have an interval with some sharps or flats, we will ignore the sharps and flats to begin with and pretend that both notes are natural. Then we will identify that interval. Once we've done that we can add the sharps and flats one by one. As we add each accidental, we will identify the interval, using the chart above, and taking into account whether the accidental makes the interval bigger or smaller.

Remember that any sharps and flats only affect the interval quality, never the number. The number depends only on the stave positions or letter names of the notes.

Let's look at a simple exmaple.

A-C♯ is a third A-C is a minor 3rd A-C♯ is one semitone bigger, so it is a major 3rd

Here's another example

C♯-E♭ is a third C-E is a major 3rd C♯-E is one semitone smaller, so it is a minor 3rd C♯-E♭ is one semitone smaller, so it is a diminished 3rd

Remember that a sharp raises a pitch by a semitone, so if it's on the higher note it will make the interval bigger, but if it's on the lower note it will make it smaller.

Notice that it wouldn't matter if we added the flat first and then the sharp. That would give us:

  • C-E is a major third.
  • C-E♭ is a minor 3rd.
  • C♯-E♭ is a diminished 3rd.

If you identify the intervals correctly at every stage, you will get the same result.

Congratulations! If you have understood everything up to this point, you can now identify any interval up to an octave! Make sure you get a lot of practice identifying intervals. It will help you understand many other topics in music theory.
