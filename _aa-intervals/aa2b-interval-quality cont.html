learning to identify intervals in the C major scale

In this section we will learn to fully identify all the intervals between any two notes from the C major scale. By the end of this section you will be able to identify any interval between any notes, as long as they don't have any sharps or flats.

Below are 2 octaves of the C major scale. The semitones (E-F and B-C) are highlighted. All the other steps are tones. A step of a tone is bigger than a step of a semitone. Everything in this section will be based around this, so it's important that you are familiar with it.

Let's go through all the intervals.

Seconds

We can see that all the seconds are tones, except for E-F and B-C which are semitones. As we know from the previous section, seconds can be major or minor. So the tones are major, and the semitones are minor.

The animation below shows for all the seconds, whther they are major or minor. As we move our two selected notes, we see that if the semitone falls inside the interval, it is minor. Otherwise it is major.

Thirds

We can see that some thirds are made up of two tones, and others are made up of a tone and a semitone. The ones with the semitone are smaller so they are minor.

In the animation below, you can see that if the semitone falls between the two notes, then it is a minor third.

Fourths

Remember that fourths can't be major or minor. Instead, they are perfect.

Most fourths contain two tones and a semitone. Those are perfect. The only exception is F-B, which doesn't contain a semitone (it contains 3 tones) so it's augmented.

Fifths

Like fourths, fifths are also perfect rather than major or minor. Most fifths contain one semitone. The only exception is B-F, which contains two. Because it contains an extra semitone, it is smaller than the perfect ones, so it is diminished. This might seem confusing, but remember that when we say it contains an extra semitone, we don't mean that the interval is a semitone bigger. We mean that one of the steps is a semitone rather than a tone, so the interval is actually a semitone smaller.

Sixths

All the sixths contain at least one semitone, but some contain two. The ones that contain two are smaller, so they are minor. The ones that only contain one are major.

Sevenths

Similar to sixths, sevenths are major if they contain one semitone, minor if they contain two.

Unisons and Octaves

All the unisons and all the octaves are the same size. All the unisons are between two of the same note, and all the octaves contain two semitones. They are all perfect (remember: never major or minor).

Summary

Congratulations! You are now able to identify any interval between natural notes. This is a very important step, so make sure you understand everything and you give yourself enough practice before tackling intervals with sharps and flats.

By the end of the next section, you will be able to identify any interval by using what you've learned in this section and in the previous one.
